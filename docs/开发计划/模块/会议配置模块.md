## 一、模块概述

会议配置模块负责管理会议的基本信息、日程安排、演讲专家和住宿推荐等会议相关数据。用户在门户端可以查看会议信息，管理员在后台可以管理这些内容。

### 1.1 功能范围

| 功能 | 说明 |
|------|------|
| 会议信息管理 | 会议标题、时间、地点、描述、横幅图片 |
| 日程管理 | 按日期组织的会议日程、日程项管理 |
| 专家管理 | 会议专家信息、主旨演讲嘉宾标记 |
| 酒店管理 | 推荐酒店信息、房型管理 |
| 图片上传 | 会议横幅、专家头像、酒店图片 |

### 1.2 模块依赖

```
会议配置模块
    ├─ 依赖：基础层（Entity、Mapper、Result）、文件上传服务
    ├─ 被依赖：门户端页面
    └─ 外部依赖：无
```

### 4.2 关键代码示例

#### ConferenceService - 获取发布的会议信息

```java
@Service
@RequiredArgsConstructor
public class ConferenceServiceImpl implements ConferenceService {

    private final ConferenceInfoMapper mapper;
    private final RedisTemplate<String, Object> redisTemplate;

    @Override
    public ConferenceInfo getPublishedConference() {
        String cacheKey = "conference:info";
        ConferenceInfo conference = (ConferenceInfo) redisTemplate.opsForValue().get(cacheKey);

        if (conference == null) {
            LambdaQueryWrapper<ConferenceInfo> wrapper = new LambdaQueryWrapper<>();
            wrapper.eq(ConferenceInfo::getIsPublished, 1);
            conference = mapper.selectOne(wrapper);

            if (conference != null) {
                redisTemplate.opsForValue().set(cacheKey, conference, 1, TimeUnit.HOURS);
            }
        }
        return conference;
    }

    @Override
    @CacheEvict(value = "conference", allEntries = true)
    public void saveOrUpdate(ConferenceInfo conference) {
        if (conference.getId() == null) {
            mapper.insert(conference);
        } else {
            mapper.updateById(conference);
        }
    }
}
```

#### ScheduleService - 获取日程列表（分组）

```java
@Service
@RequiredArgsConstructor
public class ScheduleServiceImpl implements ScheduleService {

    private final ScheduleMapper scheduleMapper;
    private final ScheduleItemMapper itemMapper;
    private final ExpertMapper expertMapper;

    @Override
    public List<ScheduleGroupVO> getScheduleList(LocalDate startDate, LocalDate endDate) {
        LambdaQueryWrapper<Schedule> wrapper = new LambdaQueryWrapper<>();
        wrapper.orderByAsc(Schedule::getDate, Schedule::getSortOrder);

        if (startDate != null) {
            wrapper.ge(Schedule::getDate, startDate);
        }
        if (endDate != null) {
            wrapper.le(Schedule::getDate, endDate);
        }

        List<Schedule> schedules = scheduleMapper.selectList(wrapper);

        // 按日期分组
        Map<LocalDate, List<ScheduleVO>> grouped = schedules.stream()
            .collect(Collectors.groupingBy(
                Schedule::getDate,
                LinkedHashMap::new,
                Collectors.mapping(this::toVO, Collectors.toList())
            ));

        // 转换为响应格式
        return grouped.entrySet().stream()
            .map(entry -> new ScheduleGroupVO(
                entry.getKey().toString(),
                entry.getValue()
            ))
            .collect(Collectors.toList());
    }

    private ScheduleVO toVO(Schedule schedule) {
        ScheduleVO vo = BeanUtil.copyProperties(schedule, ScheduleVO.class);
        vo.setDate(schedule.getDate().toString());

        // 查询日程项
        LambdaQueryWrapper<ScheduleItem> wrapper = new LambdaQueryWrapper<>();
        wrapper.eq(ScheduleItem::getScheduleId, schedule.getId());
        wrapper.orderByAsc(ScheduleItem::getSortOrder, ScheduleItem::getTime);

        List<ScheduleItem> items = itemMapper.selectList(wrapper);
        vo.setItems(items.stream()
            .map(this::itemToVO)
            .collect(Collectors.toList()));

        return vo;
    }

    private ScheduleItemVO itemToVO(ScheduleItem item) {
        ScheduleItemVO vo = BeanUtil.copyProperties(item, ScheduleItemVO.class);
        vo.setTime(item.getTime().toString());

        // 查询演讲人信息
        if (item.getSpeakerId() != null) {
            Expert expert = expertMapper.selectById(item.getSpeakerId());
            if (expert != null) {
                vo.setSpeaker(new ExpertSimpleVO(
                    expert.getId(),
                    expert.getName(),
                    expert.getTitle(),
                    expert.getOrganization(),
                    expert.getAvatarUrl()
                ));
            }
        }
        return vo;
    }
}
```

#### ExpertService - 获取专家列表

```java
@Service
@RequiredArgsConstructor
public class ExpertServiceImpl implements ExpertService {

    private final ExpertMapper mapper;

    @Override
    public ExpertListVO getExpertList(Boolean isKeynoteOnly) {
        List<Expert> experts;

        if (Boolean.TRUE.equals(isKeynoteOnly)) {
            // 仅主旨演讲嘉宾
            LambdaQueryWrapper<Expert> wrapper = new LambdaQueryWrapper<>();
            wrapper.eq(Expert::getIsKeynote, 1);
            wrapper.orderByAsc(Expert::getSortOrder);
            experts = mapper.selectList(wrapper);
        } else {
            // 全部专家
            LambdaQueryWrapper<Expert> wrapper = new LambdaQueryWrapper<>();
            wrapper.orderByDesc(Expert::getIsKeynote)
                   .orderByAsc(Expert::getSortOrder);
            experts = mapper.selectList(wrapper);
        }

        // 分组
        List<ExpertVO> keynoteExperts = new ArrayList<>();
        List<ExpertVO> speakers = new ArrayList<>();

        for (Expert expert : experts) {
            ExpertVO vo = BeanUtil.copyProperties(expert, ExpertVO.class);
            if (expert.getIsKeynote() == 1) {
                keynoteExperts.add(vo);
            } else {
                speakers.add(vo);
            }
        }

        return new ExpertListVO(keynoteExperts, speakers);
    }
}
```

## 七、开发注意事项

1. **图片上传**：
   - 限制图片大小（最大 2MB）
   - 支持常见格式（JPG、PNG）
   - 生成缩略图以提高加载速度

2. **数据验证**：
   - 会议结束日期不能早于开始日期
   - 日程日期必须在会议日期范围内
   - 酒店星级限制为 1-5

3. **缓存策略**：
   - 会议信息缓存 1 小时
   - 日程列表缓存 30 分钟
   - 更新操作后清除对应缓存

4. **排序规则**：
   - 日程按日期、排序号排序
   - 日程项按时间、排序号排序
   - 专家主旨嘉宾优先，然后按排序号
