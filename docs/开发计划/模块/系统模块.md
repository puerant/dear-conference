## 一、模块概述

系统模块负责管理平台的基础配置信息，包括邮箱配置、凭证配置、支付配置、定时任务配置和文件管理。这些配置在系统启动时加载，支持动态更新，无需重启服务。

### 1.1 功能范围

| 功能 | 说明 |
|------|------|
| 邮箱配置 | SMTP 服务器配置、测试发送 |
| 凭证配置 | 凭证模板、水印、有效期设置 |
| 支付配置 | 多渠道支付配置、沙盒模式 |
| 定时任务配置 | 订单超时时间、检查间隔 |
| 文件管理 | 图片/PDF/Word/PPT 文件上传、管理、预览 |

### 1.2 模块依赖

```
系统模块
    ├─ 依赖：基础层（Entity、Mapper、Result）
    ├─ 被依赖：订单模块（邮件发送）、凭证模块（生成凭证）、支付模块（支付渠道）
    └─ 外部依赖：SMTP 服务器、支付网关
```

#### EmailConfigService - 加密处理

```java
@Service
@RequiredArgsConstructor
public class EmailConfigServiceImpl implements EmailConfigService {

    private final EmailConfigMapper emailConfigMapper;
    private final AesUtil aesUtil;
    private final EmailLogService emailLogService;

    @Override
    public void saveOrUpdate(EmailConfig config) {
        // 密码加密存储
        if (StringUtils.isNotBlank(config.getPassword())) {
            config.setPassword(aesUtil.encrypt(config.getPassword()));
        }
        emailConfigMapper.insertOrUpdate(config);
    }

    @Override
    public void sendTestEmail(String toEmail) {
        EmailConfig config = getEnabledConfig();
        if (config == null) {
            throw new BusinessException(ErrorCode.EMAIL_CONFIG_NOT_ENABLED);
        }

        try {
            JavaMailSender mailSender = buildMailSender(config);
            SimpleMailMessage message = new SimpleMailMessage();
            message.setFrom(config.getFromEmail());
            message.setTo(toEmail);
            message.setSubject("会务通系统 - 邮箱配置测试");
            message.setText("这是一封测试邮件，如果您收到此邮件，说明邮箱配置正确。");

            mailSender.send(message);

            // 记录日志
            emailLogService.log(toEmail, "测试邮件", "test", true, null);
        } catch (Exception e) {
            emailLogService.log(toEmail, "测试邮件", "test", false, e.getMessage());
            throw new BusinessException(ErrorCode.EMAIL_SEND_FAILED, e.getMessage());
        }
    }

    private JavaMailSender buildMailSender(EmailConfig config) {
        JavaMailSenderImpl mailSender = new JavaMailSenderImpl();
        mailSender.setHost(config.getHost());
        mailSender.setPort(config.getPort());
        mailSender.setUsername(config.getUsername());
        mailSender.setPassword(aesUtil.decrypt(config.getPassword()));
        mailSender.setProtocol("smtp");
        mailSender.setDefaultEncoding("UTF-8");

        Properties props = new Properties();
        props.put("mail.smtp.auth", "true");
        if (config.getSslEnabled()) {
            props.put("mail.smtp.ssl.enable", "true");
        }
        mailSender.setJavaMailProperties(props);

        return mailSender;
    }
}
```

#### OrderTaskConfigService - 动态配置

```java
@Service
@RequiredArgsConstructor
public class OrderTaskConfigServiceImpl implements OrderTaskConfigService {

    private final OrderTaskConfigMapper configMapper;
    private final RedisTemplate<String, Object> redisTemplate;

    @Override
    public OrderTaskConfig getConfig() {
        // 先从缓存读取
        String cacheKey = "system:order_task_config";
        OrderTaskConfig config = (OrderTaskConfig) redisTemplate.opsForValue().get(cacheKey);

        if (config == null) {
            config = configMapper.selectOne(null);
            if (config == null) {
                config = new OrderTaskConfig();
                config.setTimeoutMinutes(30);
                config.setCheckIntervalMinutes(5);
            }
            redisTemplate.opsForValue().set(cacheKey, config, 1, TimeUnit.HOURS);
        }
        return config;
    }

    @Override
    public void updateConfig(OrderTaskConfig config) {
        configMapper.updateById(config);
        // 清除缓存
        redisTemplate.delete("system:order_task_config");
    }
}
```

### 4.3 配置加密工具（AesUtil）

```java
@Component
public class AesUtil {

    @Value("${app.config.secret:default-secret-key-16bytes}")
    private String secretKey;

    private static final String ALGORITHM = "AES/CBC/PKCS5Padding";

    public String encrypt(String plainText) {
        try {
            Cipher cipher = Cipher.getInstance(ALGORITHM);
            SecretKeySpec keySpec = new SecretKeySpec(secretKey.getBytes(StandardCharsets.UTF_8), "AES");
            IvParameterSpec ivSpec = new IvParameterSpec(secretKey.getBytes(StandardCharsets.UTF_8), 0, 16);
            cipher.init(Cipher.ENCRYPT_MODE, keySpec, ivSpec);
            byte[] encrypted = cipher.doFinal(plainText.getBytes(StandardCharsets.UTF_8));
            return Base64.getEncoder().encodeToString(encrypted);
        } catch (Exception e) {
            throw new RuntimeException("加密失败", e);
        }
    }

    public String decrypt(String cipherText) {
        try {
            Cipher cipher = Cipher.getInstance(ALGORITHM);
            SecretKeySpec keySpec = new SecretKeySpec(secretKey.getBytes(StandardCharsets.UTF_8), "AES");
            IvParameterSpec ivSpec = new IvParameterSpec(secretKey.getBytes(StandardCharsets.UTF_8), 0, 16);
            cipher.init(Cipher.DECRYPT_MODE, keySpec, ivSpec);
            byte[] decrypted = cipher.doFinal(Base64.getDecoder().decode(cipherText));
            return new String(decrypted, StandardCharsets.UTF_8);
        } catch (Exception e) {
            throw new RuntimeException("解密失败", e);
        }
    }
}
```

#### SystemFileService - 文件上传与处理

```java
@Service
@RequiredArgsConstructor
public class SystemFileServiceImpl implements SystemFileService {

    private final SystemFileMapper fileMapper;
    private final FileStorageService fileStorageService;

    // 文件大小限制（字节）
    private static final long MAX_IMAGE_SIZE = 5 * 1024 * 1024;        // 5MB
    private static final long MAX_DOCUMENT_SIZE = 20 * 1024 * 1024;   // 20MB
    private static final long MAX_PRESENTATION_SIZE = 30 * 1024 * 1024; // 30MB

    // 支持的文件类型
    private static final Set<String> IMAGE_EXTENSIONS = Set.of("jpg", "jpeg", "png", "gif", "webp");
    private static final Set<String> DOCUMENT_EXTENSIONS = Set.of("pdf", "doc", "docx");
    private static final Set<String> PRESENTATION_EXTENSIONS = Set.of("ppt", "pptx");

    @Override
    public SystemFile uploadFile(MultipartFile file, String fileCategory, Long categoryId) {
        // 校验文件
        validateFile(file);

        // 计算文件MD5
        String md5 = DigestUtils.md5DigestAsHex(file.getInputStream());

        // 检查是否已存在相同文件（去重）
        SystemFile existingFile = fileMapper.selectOne(
            new LambdaQueryWrapper<SystemFile>()
                .eq(SystemFile::getMd5, md5)
                .eq(SystemFile::getIsDeleted, 0)
        );
        if (existingFile != null) {
            return existingFile; // 返回已有文件
        }

        // 存储文件
        String fileUrl = fileStorageService.store(file);

        // 保存文件记录
        SystemFile systemFile = new SystemFile();
        systemFile.setFileName(file.getOriginalFilename());
        systemFile.setFilePath(fileStorageService.getFilePath(file));
        systemFile.setFileUrl(fileUrl);
        systemFile.setFileType(determineFileType(file.getOriginalFilename()));
        systemFile.setFileExtension(getFileExtension(file.getOriginalFilename()));
        systemFile.setMimeType(file.getContentType());
        systemFile.setFileSize(file.getSize());
        systemFile.setFileCategory(fileCategory);
        systemFile.setCategoryId(categoryId);
        systemFile.setStorageType(fileStorageService.getStorageType());
        systemFile.setMd5(md5);

        // 设置上传用户信息
        HttpServletRequest request = ((ServletRequestAttributes) RequestContextHolder.getRequestAttributes()).getRequest();
        Object user = request.getAttribute("currentUser");
        if (user != null) {
            systemFile.setUploadUserId(((User) user).getId());
            systemFile.setUploadUserName(((User) user).getName());
        }

        fileMapper.insert(systemFile);
        return systemFile;
    }

    @Override
    public void deleteFile(Long id) {
        SystemFile file = fileMapper.selectById(id);
        if (file == null) {
            throw new BusinessException(ErrorCode.FILE_NOT_FOUND);
        }

        // 软删除
        file.setIsDeleted(1);
        fileMapper.updateById(file);

        // TODO: 定时任务清理物理文件
    }

    @Override
    public PageResult<SystemFile> getFileList(SystemFileQuery query) {
        LambdaQueryWrapper<SystemFile> wrapper = new LambdaQueryWrapper<>();

        if (StringUtils.isNotBlank(query.getFileType())) {
            wrapper.eq(SystemFile::getFileType, query.getFileType());
        }
        if (StringUtils.isNotBlank(query.getFileCategory())) {
            wrapper.eq(SystemFile::getFileCategory, query.getFileCategory());
        }
        if (query.getCategoryId() != null) {
            wrapper.eq(SystemFile::getCategoryId, query.getCategoryId());
        }
        if (query.getStartDate() != null) {
            wrapper.ge(SystemFile::getCreatedAt, query.getStartDate());
        }
        if (query.getEndDate() != null) {
            wrapper.le(SystemFile::getCreatedAt, query.getEndDate());
        }
        if (StringUtils.isNotBlank(query.getKeyword())) {
            wrapper.like(SystemFile::getFileName, query.getKeyword());
        }

        wrapper.eq(SystemFile::getIsDeleted, 0);
        wrapper.orderByDesc(SystemFile::getCreatedAt);

        Page<SystemFile> page = fileMapper.selectPage(
            new Page<>(query.getPage(), query.getPageSize()),
            wrapper
        );

        return new PageResult<>(page.getRecords(), page.getTotal(), query.getPage(), query.getPageSize());
    }

    private void validateFile(MultipartFile file) {
        if (file.isEmpty()) {
            throw new BusinessException(ErrorCode.FILE_EMPTY);
        }

        String filename = file.getOriginalFilename();
        String extension = getFileExtension(filename).toLowerCase();

        // 验证文件类型
        String fileType = determineFileType(filename);
        long maxSize = getMaxFileSizeByType(fileType);
        if (file.getSize() > maxSize) {
            throw new BusinessException(ErrorCode.FILE_TOO_LARGE);
        }

        // 验证扩展名
        Set<String> allowedExtensions = getAllowedExtensions(fileType);
        if (!allowedExtensions.contains(extension)) {
            throw new BusinessException(ErrorCode.FILE_TYPE_NOT_ALLOWED);
        }
    }

    private String determineFileType(String filename) {
        String extension = getFileExtension(filename).toLowerCase();
        if (IMAGE_EXTENSIONS.contains(extension)) {
            return "image";
        } else if (DOCUMENT_EXTENSIONS.contains(extension)) {
            return "document";
        } else if (PRESENTATION_EXTENSIONS.contains(extension)) {
            return "presentation";
        }
        return "unknown";
    }

    private long getMaxFileSizeByType(String fileType) {
        return switch (fileType) {
            case "image" -> MAX_IMAGE_SIZE;
            case "document" -> MAX_DOCUMENT_SIZE;
            case "presentation" -> MAX_PRESENTATION_SIZE;
            default -> MAX_IMAGE_SIZE;
        };
    }

    private Set<String> getAllowedExtensions(String fileType) {
        return switch (fileType) {
            case "image" -> IMAGE_EXTENSIONS;
            case "document" -> DOCUMENT_EXTENSIONS;
            case "presentation" -> PRESENTATION_EXTENSIONS;
            default -> Set.of();
        };
    }

    private String getFileExtension(String filename) {
        if (filename == null || !filename.contains(".")) {
            return "";
        }
        return filename.substring(filename.lastIndexOf(".") + 1);
    }
}
```

#### LocalFileStorageService - 本地存储实现

```java
@Service
@ConditionalOnProperty(name = "file.storage.type", havingValue = "local")
@RequiredArgsConstructor
public class LocalFileStorageService implements FileStorageService {

    @Value("${file.storage.path:/data/files}")
    private String basePath;

    @Value("${server.file-access-url:/files}")
    private String fileAccessUrl;

    @Override
    public String store(MultipartFile file) throws IOException {
        // 按日期创建目录
        String datePath = LocalDate.now().format(DateTimeFormatter.ofPattern("yyyyMMdd"));
        String fullPath = basePath + "/" + datePath;
        File dir = new File(fullPath);
        if (!dir.exists()) {
            dir.mkdirs();
        }

        // 生成唯一文件名
        String originalFilename = file.getOriginalFilename();
        String extension = originalFilename.substring(originalFilename.lastIndexOf("."));
        String filename = UUID.randomUUID().toString() + extension;

        // 保存文件
        File targetFile = new File(fullPath, filename);
        file.transferTo(targetFile);

        // 返回访问URL
        return fileAccessUrl + "/" + datePath + "/" + filename;
    }

    @Override
    public String getFilePath(MultipartFile file) {
        String datePath = LocalDate.now().format(DateTimeFormatter.ofPattern("yyyyMMdd"));
        String filename = UUID.randomUUID().toString() + getFileExtension(file.getOriginalFilename());
        return basePath + "/" + datePath + "/" + filename;
    }

    @Override
    public int getStorageType() {
        return 1; // 本地存储
    }
}
```

## 七、开发注意事项

1. **安全要求**：
   - 密码、密钥必须加密存储（AES）
   - 配置接口仅管理员可访问
   - 配置值不在前端明文展示（密码字段使用 `*` 遮挡）
   - 文件上传必须验证文件类型，防止恶意文件上传
   - 文件路径不得暴露物理存储路径

2. **缓存策略**：
   - 邮箱配置、支付配置写入缓存（1小时）
   - 配置更新后清除缓存

3. **兼容性**：
   - 邮箱配置支持 SSL/TLS
   - 支付配置支持微信和支付宝
   - 文件存储支持本地和云存储（OSS/COS）

4. **日志记录**：
   - 配置变更记录到操作日志
   - 测试邮件发送记录到邮件日志
   - 文件上传记录到系统日志

5. **文件管理**：
   - 文件上传前校验大小和类型
   - 相同文件（MD5相同）不重复存储
   - 删除文件采用软删除，物理文件定期清理
   - 支持分页查询，支持按类型/分类筛选
   - 图片文件生成缩略图（可选）

### 10.2 Bug 修复详情

#### BUG-SYS-001: Redis 缓存反序列化问题

**问题描述**：
访问 `/api/admin/system/config` 接口时报错：
```
系统异常: uri=/api/admin/system/config, message=class java.util.LinkedHashMap cannot be cast to class com.huiwutong.conference.entity.OrderTaskConfig
```

后续尝试启用 `activateDefaultTyping` 后又出现：
```
SerializationException: Could not read JSON: missing type id property '@class'
```

**原因分析**：
- `GenericJackson2JsonRedisSerializer` 在反序列化时，会将复杂对象反序列化为 `LinkedHashMap`
- 尝试启用 `activateDefaultTyping` 后，旧缓存数据缺少 `@class` 属性导致新的序列化异常
- 异常发生在 Redis 模板层，服务层的 try-catch 无法捕获

**最终修复方案**：

1. 更换缓存 key 为 `system:order_task_config:v2`，绕过旧的不兼容数据
2. 保持原 RedisConfig 序列化配置不变（不启用 activateDefaultTyping）
3. 服务层增加防御性类型检查：

```java
private static final String CACHE_KEY = "system:order_task_config:v2";

@Override
public OrderTaskConfig getConfig() {
    OrderTaskConfig config = null;

    try {
        Object cached = redisTemplate.opsForValue().get(CACHE_KEY);
        if (cached instanceof OrderTaskConfig) {
            config = (OrderTaskConfig) cached;
        } else if (cached != null) {
            // 缓存格式不兼容（如 LinkedHashMap），清除旧缓存
            redisTemplate.delete(CACHE_KEY);
        }
    } catch (Exception e) {
        // 序列化异常或其他异常，清除旧缓存
        redisTemplate.delete(CACHE_KEY);
    }

    if (config == null) {
        // 从数据库查询...
    }
    return config;
}
```

**修改文件**：
- `backend/src/main/java/com/huiwutong/conference/service/impl/OrderTaskConfigServiceImpl.java`

**验证结果**：后端编译成功，接口正常返回数据
