## 一、模块概述

日志模块负责记录系统运行过程中的重要日志信息，包括定时任务执行日志和邮件发送日志。日志可用于问题排查、审计追踪和数据分析。

### 1.1 功能范围

| 功能 | 说明 |
|------|------|
| 定时任务日志 | 记录订单超时取消、退款检查等定时任务的执行情况 |
| 邮件发送日志 | 记录所有邮件发送的结果，成功或失败 |
| 日志查询 | 支持按时间、类型、状态筛选查询 |
| 日志导出 | 支持导出为 Excel 格式 |

### 1.2 模块依赖

```
日志模块
    ├─ 依赖：基础层（Entity、Mapper、Result）
    ├─ 被依赖：订单模块（记录任务执行）、邮件服务（记录发送结果）
    └─ 外部依赖：无
```

### 4.2 关键代码示例

#### OrderTaskLogService - 记录定时任务执行

```java
@Service
@RequiredArgsConstructor
public class OrderTaskLogServiceImpl implements OrderTaskLogService {

    private final OrderTaskLogMapper logMapper;

    @Override
    public void logSuccess(String taskType, Long orderId, String orderNo, String message, long durationMs) {
        OrderTaskLog log = new OrderTaskLog();
        log.setTaskType(taskType);
        log.setOrderId(orderId);
        log.setOrderNo(orderNo);
        log.setStatus(1); // 成功
        log.setMessage(message);
        log.setExecuteTime(LocalDateTime.now());
        log.setDurationMs((int) durationMs);
        logMapper.insert(log);
    }

    @Override
    public void logFailure(String taskType, Long orderId, String orderNo, String error, long durationMs) {
        OrderTaskLog log = new OrderTaskLog();
        log.setTaskType(taskType);
        log.setOrderId(orderId);
        log.setOrderNo(orderNo);
        log.setStatus(2); // 失败
        log.setMessage("任务执行失败");
        log.setErrorDetail(error);
        log.setExecuteTime(LocalDateTime.now());
        log.setDurationMs((int) durationMs);
        logMapper.insert(log);
    }

    /**
     * 带自动记录的执行器
     */
    public void executeWithLog(String taskType, Long orderId, String orderNo, Runnable task) {
        long startTime = System.currentTimeMillis();
        try {
            task.run();
            long duration = System.currentTimeMillis() - startTime;
            logSuccess(taskType, orderId, orderNo, "执行成功", duration);
        } catch (Exception e) {
            long duration = System.currentTimeMillis() - startTime;
            logFailure(taskType, orderId, orderNo, e.getMessage(), duration);
            throw e;
        }
    }
}
```

#### OrderCancelTask - 集成日志记录

```java
@Component
@RequiredArgsConstructor
public class OrderCancelTask {

    private final OrderMapper orderMapper;
    private final TicketMapper ticketMapper;
    private final CredentialMapper credentialMapper;
    private final GroupMemberMapper groupMemberMapper;
    private final OrderTaskLogService logService;

    @Scheduled(cron = "${order.cancel.cron:0 */5 * * * ?}")
    public void cancelTimeoutOrders() {
        LocalDateTime timeoutThreshold = LocalDateTime.now().minusMinutes(30);

        List<Order> timeoutOrders = orderMapper.findTimeoutUnpaidOrders(timeoutThreshold);

        if (CollectionUtils.isEmpty(timeoutOrders)) {
            return;
        }

        for (Order order : timeoutOrders) {
            logService.executeWithLog("order_cancel", order.getId(), order.getOrderNo(), () -> {
                // 取消订单
                orderMapper.cancelOrder(order.getId());

                // 恢复库存
                ticketMapper.increaseStock(order.getTicketId(), order.getQuantity());

                // 处理凭证和团体成员
                if (order.getOrderType() == 2) {
                    // 团体票：删除成员
                    groupMemberMapper.deleteByOrderId(order.getId());
                } else {
                    // 个人票：失效凭证
                    credentialMapper.expireByOrderId(order.getId());
                }
            });
        }
    }
}
```

### 4.3 操作日志切面

```java
@Aspect
@Component
@Slf4j
@RequiredArgsConstructor
public class OperationLogAspect {

    private final OperationLogService logService;

    @Pointcut("@annotation(com.huiwutong.conference.common.annotation.OperationLog)")
    public void operationLogPointcut() {}

    @Around("operationLogPointcut()")
    public Object around(ProceedingJoinPoint point) throws Throwable {
        long beginTime = System.currentTimeMillis();
        Object result = null;
        Exception exception = null;

        try {
            result = point.proceed();
            return result;
        } catch (Exception e) {
            exception = e;
            throw e;
        } finally {
            long duration = System.currentTimeMillis() - beginTime;
            saveLog(point, result, exception, duration);
        }
    }

    private void saveLog(ProceedingJoinPoint point, Object result, Exception exception, long duration) {
        // 获取当前请求
        HttpServletRequest request = ((ServletRequestAttributes) RequestContextHolder.getRequestAttributes()).getRequest();

        OperationLog log = new OperationLog();
        log.setIp(IpUtil.getIpAddr(request));
        log.setDurationMs((int) duration);
        log.setStatus(exception == null ? 1 : 2);

        // 获取方法信息
        MethodSignature signature = (MethodSignature) point.getSignature();
        Method method = signature.getMethod();
        String className = point.getTarget().getClass().getName();
        log.setMethod(className + "." + method.getName());

        // 获取注解信息
        OperationLog operationLog = method.getAnnotation(OperationLog.class);
        log.setOperation(operationLog.value());
        log.setModule(operationLog.module());

        // 获取用户信息
        Object user = request.getAttribute("currentUser");
        if (user != null) {
            log.setUserId(((User) user).getId());
            log.setUserName(((User) user).getName());
        }

        // 请求参数（脱敏）
        Object[] args = point.getArgs();
        if (args != null && args.length > 0) {
            String params = JsonUtil.toJson(args);
            log.setParams(SensitiveUtil.mask(params));
        }

        // 错误信息
        if (exception != null) {
            log.setErrorMsg(ExceptionUtil.getMessage(exception));
        }

        logService.save(log);
    }
}
```

const formatDate = (date: string | Date) => {
  if (!date) return ''
  const d = new Date(date)
  return d.toLocaleString('zh-CN')

  handleSearch()

| TC-LOG-006 | 日志导出 | 点击导出按钮 | 下载 Excel 文件，包含日志数据 |
| TC-LOG-007 | 日志统计 | 访问统计接口 | 返回成功/失败数量和成功率 |

## 七、开发注意事项

1. **敏感信息保护**：
   - 日志中不得记录密码、密钥等敏感信息
   - 错误堆栈信息需要脱敏处理

2. **性能优化**：
   - 使用 `@Async` 异步记录日志，不影响主业务
   - 定时任务日志记录不应阻塞任务执行

3. **日志归档**：
   - 建议定期归档历史日志（保留 90 天）
   - 归档日志可迁移到其他存储

4. **查询优化**：
   - 建立必要的索引
   - 大量数据查询时限制查询范围
