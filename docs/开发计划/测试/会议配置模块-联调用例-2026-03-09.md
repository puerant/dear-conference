> 日期：2026-03-09
> 范围：Portal + Admin + Backend（会议配置模块）
> 工具：Chrome DevTools（Network / Console / Application）

## 1. 联调前准备

1. 后端启动：`http://localhost:8080`
2. Admin 启动：`http://localhost:3001`
3. Portal 启动：`http://localhost:5173`
4. Chrome DevTools 打开：
- `Network`：勾选 `Preserve log`、`Disable cache`
- `Console`：观察报错和告警
- `Application > Local Storage`：检查 token 与用户信息

2. 测试日期：建议使用会议期间 `2026-06-15 ~ 2026-06-17`

- `jpg/png` 小图（< 2MB）
- 超限图（> 2MB）
