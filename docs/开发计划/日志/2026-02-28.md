| `available` 字段为负数 | sold_count 超过 stock | 检查 `TicketServiceImpl` 库存扣减逻辑是否有并发保护 |
| 支付后 `sold_count` 未变 | 支付回调未触发库存扣减 | 检查 `PaymentController.callback()` 是否调用 `ticketService.incrementSoldCount()` |
| 支付后 `credentials` 表无记录 | 凭证生成未触发 | 检查支付回调是否调用 `credentialService.generate()`，查看后端日志 |
| 凭证二维码为空或乱码 | 二维码数据序列化问题 | 检查 `CredentialServiceImpl` 中 QR 码生成逻辑，确认 JSON 字段完整 |
| 退款后库存未恢复 | `refundApprove` 未更新 sold_count | 检查 `RefundServiceImpl.approve()` 是否调用 `ticketService.decrementSoldCount()` |
| 退款后凭证仍有效 | 凭证状态联动未实现 | 检查退款审批逻辑中是否更新 `credentials.status = 3` |
| 定时任务未触发 | Scheduler 配置问题或任务 cron 表达式 | 检查 `@Scheduled` 注解和 Spring Scheduler 配置 |
| Admin 票务统计数据不准 | 统计 SQL 缺少 GROUP BY 或条件 | 直接对比 SQL 查询与接口返回值 |
