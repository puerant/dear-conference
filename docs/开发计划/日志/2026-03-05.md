## 二、今日问题与修复

### BUG-FILE-SQL-001：MySQL 不支持 `ADD COLUMN IF NOT EXISTS` ✅ 已修复

**现象**：
执行 `08_file_module_oss.sql` 报错：
`[42000][1064] ... near 'IF NOT EXISTS provider_id ...'`

**原因**：
当前 MySQL 版本不支持 `ALTER TABLE ... ADD COLUMN IF NOT EXISTS` 语法。

**修复方案**：
将列变更改为兼容写法：
1. 查询 `information_schema.columns` 判断字段是否存在。
2. 使用动态 SQL 执行 `ALTER TABLE ... ADD COLUMN`。

**修复结果**：
脚本可在不支持该语法的版本上重复执行，不再因该语法报错。
